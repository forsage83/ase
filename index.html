Aura Slider Engine

Is a jQuery based, layout independent, lightweight, configurable slider engine for responsive and static sliders.

IE7+, Google Chrome, Opera, Safari,
Firefox

Download

Lightweight & simple

For those who need a rugged, light and extensible base (ie7 +) only.
Determine by yourself what, where and how should look and feel.

Any content & any markup

Slide anything — text, pictures, video. Use any markup and classes.

Layout independent

Use service 'js-' classes or overriding them in to ours. Any markup you`d like.

Static & Responsive

Responsive as you make it responsive. Static if you make it static.

Flexible & modular

Определяйте все что вам нужно при инициализации.

Swipe & touch support

Support touch and swipe events from jQuery mobile library.

Slide horiz

Один элемент полностью уходит за область видимости. Перемещение происходит в горизонтальной плосткости.

Slide vert

Один элемент полностью уходит за область видимости. Перемещение происходит в вертикальной плосткости.

Fade slide

Один элемент полностью уходит за область видимости при этом теряя прозрачность. Перемещение происходит в горизонтальной плосткости.

Grow

Пропорциональное уменьшение элемента с увеличением прозрачности.

Grow fade

Переход элемента из увеличенного состояния с потерянной прозрачностью к исходным размерам.

Skew

Элемент появляется переходя от скошенного состояния к горизонтальному.

Twirl

Элемент появляется прокручиываясь по оси x в плоскости z.

Fly

Элемент появляется переходя от уменьшенной высоты к своим исходным размерам, при этом происходит увеличение прозрачности. Эффект напоминающий включение старого телевизора.

Extrude

Эффект напоминающий вытеснение одного элемента другим. Изменение размеров уходяшего элемнта с синхронным изменением позиции по x приходящего элемента.

Leaf over

Изменение перспективы уходящего элемента с синхронным изменением позиции по x появляющегося элемента.

onMove

Animation/style active number/dot

Default: {backgroundColor:'#cc3333', color:'#fff'}

Accept: array

Example: $('.box_skitter_large').skitter({animateNumberActive: {backgroundColor:'#000', color:'#ccc'}});z
